# SublimeTextSettings
Сниппеты, плагины и настройки для Sublime Text 3

Плагины:

- emmet
- autofilename
- brackethighliter
- colorhelper
- sublimecodeintel

Настройки:

{
	"color_scheme": "Packages/Color Scheme - Default/Monokai.sublime-color-scheme",
	"draw_white_space": "all",
	"font_size": 12,
	"ignored_packages":
	[
		"Vintage"
	],
	"theme": "Default.sublime-theme"
}
